Что такое CMS

CMS— это система управления контентом сайта. На профессиональном жаргоне CMS ещё называют «движок сайта». По данным агентства W3Techs на CMS работает больше половины сайтов в интернете.

Однако, для того чтобы сделать сайт с помощью CMS потребуется помошь програмиста, дизайнера верстальщика. И капиьаловложения.

Какие бывают cms

Бывают различные формы управления контентом: для интернет магазинов, для блогов, для форумов и т.д.

Примеры CMS

Примеры популярных CMS: WordPress, Joomla!, OpenCart.
